Настройка клиента на Linux



Для настройки OpenVPN-клиента должен быть архив, в котором находятся 4 файла: сертификат CA, ключ и сертификат пользователя OpenVPN и конфигурационный файл.


Распакуйте этот архив в каталог /etc/openvpn


Запустите службу OpenVpn - sudo service openvpn start


После запуска службы OpenVPN, в списке устройств должен появиться новый сетевой адаптер tap


Для проверки соединения с OpenVPN-сервером можно воспользоваться командой ping 10.8.0.1


Также, убедиться в установлении соединения с сервером можно посмотрев файл /etc/openvpn/openvpn.log
