Управление подключенными клиентами



Для управления подключенными клиентами наберите в терминале команду telnet localhost 7777


Для отображения списка всех подключений введите команду status


Чтобы отключить нужного вам пользователя, введите kill имя_клиента (в примере - client_linux). Сеанс будет отключен до перезапуска OpenVPN-сервера.


Для отображения списка всех возможных команд введите help
